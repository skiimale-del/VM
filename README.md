# Valuebets (VM) — Pinnacle baseline vs svenska bolag

Webbdashboard som visar värdespel genom att jämföra Pinnacles "true probs" (vig-borttaget)
mot odds från svenska bolag via The Odds API. Marknader: 1X2, Asian Handicap, Over/Under.

## Snabbstart (Streamlit Community Cloud)
1) Skapa ett GitHub-repo med filerna: app.py, requirements.txt, README.md
2) Gå till [streamlit.io](https://streamlit.io) → Sign in → "Deploy an app"
3) Välj ditt repo/branch och filen `app.py`.
